# WebAccesibilidad
En este proyecto he creado una web utilizando HTML, CSS y Bootstrap. La web habla de la accesibilidad web, de los distintos niveles de esta misma y de como conseguir que una web sea accesible. 

-Autor: Manuel Pérez García

-Objetivo: El objetivo de esta práctica es el de crear una web responsive con el uso de bootstrap en la que hable de la accesibilidad web, de buenas prácticas para conseguirla y los distintos niveles de esta misma.

-Estructura de carpetas y archivos: En la carpeta del trabajo encontramos los dos ficheros html (página principal y secundaria), y el fichero css que contiene los estilos de la web. También encontraremos una carpeta multimedia con las imágenes y los iconos. Por último encontraremos la guía de estilos, que será un documento pdf en el que nos encontraremos una muestra de los estilos, tipografías y colores de la web.

-Tecnologías usadas: HTML, CSS y Bootstrap.
